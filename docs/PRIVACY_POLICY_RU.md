# Политика конфиденциальности MeshCore Open

**Последнее обновление:** 11 января 2026 г.

## Введение

Repachat("Приложение") - это приложение Flutter с открытым исходным кодом для взаимодействия с сетевыми устройствами MeshCore LoRa. Эта политика конфиденциальности объясняет, как приложение обрабатывает вашу информацию.

## Сбор данных

### Данные, КОТОРЫЕ мы НЕ собираем

Repachat** не работает**:
- Собирать личную информацию
- Отправка данных на внешние серверы (за исключением запросов на отображение фрагментов карты)
- Отслеживать ваше использование или поведение
- Пользоваться аналитическими сервисами
- Требовать создания учетной записи
- Передавать любые данные третьим лицам

### Данные, хранящиеся локально на вашем устройстве

Приложение хранит следующие данные ** только локально на вашем устройстве**:

- ** Сообщения**: Сообщения чата, отправленные и полученные через mesh-сеть
- **Контакты**: Имена и идентификаторы контактов в mesh-сети
- ** Настройки приложения **: Ваши предпочтения (тема, язык, настройки уведомлений)
- ** Настройки канала **: Настройка каналов ячеистой сети
- ** История сообщений **: История путей для маршрутизации сообщений
- **Журналы отладки **: Дополнительные журналы отладки BLE и приложений (если они включены пользователем)
- ** Кэшированные листы карты **: Автономные картографические данные для функции отображения

Все локально сохраненные данные остаются на вашем устройстве и никогда не передаются нам или какой-либо третьей стороне.

## Разрешения

Для работы приложения требуются определенные разрешения устройства:

### Разрешения Bluetooth
- ** BLUETOOTH, BLUETOOTH_ADMIN ** (Android 11 и ниже)
- **BLUETOOTH_SCAN, BLUETOOTH_CONNECT, BLUETOOTH_ADVERTISE** (Android 12+)

Эти разрешения используются исключительно для обнаружения аппаратных устройств MeshCore и взаимодействия с ними через Bluetooth с низким энергопотреблением (BLE).

### Разрешение на определение местоположения
- **ДОСТУП К ТОЧНОМУ МЕСТОПОЛОЖЕНИЮ, ДОСТУП К ШИРОКОМУ МЕСТОПОЛОЖЕНИЮ**

Требуется Android для сканирования BLE на Android 11 и ниже. Приложение не отслеживает и не сохраняет ваше местоположение. Данные о местоположении могут быть дополнительно переданы по сотовой сети, если вы решите включить функции совместного использования местоположения.

### Разрешение на доступ в Интернет
- **ИНТЕРНЕТ**

Используется только для загрузки фрагментов карт с серверов OpenStreetMap при использовании функции карты. Личные данные не передаются.

### Разрешение на уведомление
- ** POST_NOTIFICATIONS** (Android 13+)

Используется для отображения уведомлений о входящих сообщениях, когда приложение работает в фоновом режиме.

### Разрешения фоновой службы
- **FOREGROUND_SERVICE, FOREGROUND_SERVICE_CONNECTED_DEVICE, WAKE_LOCK**

Используется для поддержания соединения BLE с вашим устройством MeshCore, когда приложение работает в фоновом режиме.

## Сторонние сервисы

### Плитки карты
Приложение использует серверы листов OpenStreetMap для отображения карт. При просмотре карт серверу листов может быть виден IP-адрес вашего устройства. Другие данные не передаются. Дополнительную информацию смотрите в разделе [Политика конфиденциальности OpenStreetMap](https://wiki.osmfoundation.org/wiki/Privacy_Policy).

### Поиск GIF-файлов (Giphy)
Приложение включает функцию выбора GIF-файлов, разработанную Giphy. При использовании функции поиска GIF-файлов:
- Ваши поисковые запросы отправляются на API-серверы Giphy
- Giphy видит IP-адрес вашего устройства
- Giphy может собирать данные об использовании в соответствии со своей политикой конфиденциальности

Поиск в формате GIF не является обязательным и активируется только тогда, когда вы решите его использовать. Смотрите [Политику конфиденциальности Giphy](https://support.giphy.com/hc/en-us/articles/360032872931-GIPHY-Privacy-Policy) для получения дополнительной информации о том, как они обрабатывают данные.

## Сетевые коммуникации

Сообщения, отправленные через mesh-сеть MeshCore, передаются по радиочастотам на другие устройства mesh. Само приложение не контролирует эти коммуникации, а только обеспечивает соединение между вашим мобильным устройством и оборудованием MeshCore.

## Безопасность данных

Все данные хранятся локально на вашем устройстве с использованием стандартных механизмов хранения Flutter / Android. Приложение не использует дополнительное шифрование для локально сохраненных данных, помимо того, что обеспечивает операционная система.

## Конфиденциальность детей

Приложение сознательно не собирает никакой личной информации от детей младше 13 лет.
